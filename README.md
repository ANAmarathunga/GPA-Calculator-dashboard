# 🎯 Academic GPA Dashboard & Target Advisor

A fully responsive, production-ready **Academic GPA Calculator and Strategic Roadmap Dashboard** built with **React.js**. This application goes beyond basic calculations by allowing students to organize records semester-by-semester, securely cache historical metrics, and forecast exam targets mathematically based on their upcoming workloads.

---

## 🔥 Core Features

* **📅 Semester-Wise Segmentation:** Add, manage, and isolate curriculum data into distinct semester blocks. Each container accurately outputs native semester-specific GPA values.
* **🏆 Global CGPA Aggregator:** Dynamically computes global cumulative grade points and displays the student's final honors classification degree status.
* **🔮 Target Progress Advisor (Advanced Forecast):** Input a desired target GPA and the number of remaining modules. The dashboard assumes a standard 3-credit baseline configuration per module to mathematically evaluate and recommend a concrete strategic grade requirement roadmap.
* **💾 Local Device Persistent Caching:** Utilizes modern browser `localStorage` hooks to seamlessly retain active academic tracking datasets across unexpected power drops or forced page reloads.
* **📥 Print & Local Export Engine:** Native integration with system print engines to produce perfectly formatted, page-break-optimized digital PDF report cards without broken structural lines.
* **🗑️ Structural State Purging:** Features prompt-guarded database clearing routines to factory reset application instances smoothly.
